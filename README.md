# Stock Market ETL & Risk Dashboard

An end-to-end Data Pipeline that automates the extraction, processing, and visualization of stock market data.

## 🛠️ Technologies Used
- **Python (Pandas, yfinance)**: Data Extraction and Transformation (ETL).
- **Google Cloud BigQuery**: Cloud Data Warehousing.
- **GitHub Actions**: CI/CD Automation for daily data updates.
- **Looker Studio**: Interactive Live Dashboard for risk analysis.
